# React + TypeScript + Vite

A modern frontend web application built with **React**, **TypeScript**, and **Vite**, focused on fast development, strong type safety, and optimized performance.

---

## Tech Stack

- **React** – UI library
- **TypeScript** – Static type checking
- **Vite** – Lightning-fast build tool
- **Node.js** – Runtime environment
- **Git & GitHub** – Version control

---

## Installation & Setup

Make sure you have **Node.js v20+** installed.

### Clone the repository

```bash
git clone https://github.com/darrylemiles/portfolio.git
```
